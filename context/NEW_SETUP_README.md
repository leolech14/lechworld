## Quick Start (15 minutes total)

### Step 1: Create Supabase Project
1. Go to https://supabase.com
2. Create new project "lechworld"
3. Save the password you create!

### Step 2: Run Setup
```bash
cd /Users/lech/lechworld
./setup-new-supabase.sh
```

Follow the prompts - the script handles everything:
- Configures environment
- Creates database tables
- Deploys to Fly.io
- Migrates all your data
- Goes live!

## Your Data is Safe
